# FinAPI - API Financeira

## Requisitos

- [x] Deve ser possível criar conta
- [x] Deve ser possível buscar o extrato bancário do cliente
- [ ] Deve ser possível realizar um depósito
- [ ] Deve ser possível realizar um saque
- [ ] Deve ser possível buscar o extrato bancário do cliente por data
- [ ] Deve ser possível atualizar os dados da conta
- [ ] Deve ser possível obter os dados da conta
- [ ] Deve ser possível deletar a conta

## Regras de Negócio

- [x] Não deve ser possível cadastrar um CPF repetido
- [x] Não deve ser possível buscar extrato de uma conta inexistente
- [ ] Não deve ser possível realizar o deposito de uma conta inexistente
- [ ] Não deve ser possível realizar o saque de uma conta inexistente
- [ ] Não deve ser possível excluir uma conta inexistente
- [ ] Não deve ser possível realizar o saque quando o saldo não for o suficiente
